About Stellar

Stellar is an application designed to help users manage and support their TikTok-related activities, such as authentication, account connection, and application features that require TikTok integration.

Connection with TikTok

Stellar connects with TikTok to allow secure login, access permitted account information, and enable features that require TikTok data. The application only accesses data authorized by the user and follows TikTok Developer policies.
