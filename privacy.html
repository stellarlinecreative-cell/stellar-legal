Privacy Policy

Last Updated: January 2025

This Privacy Policy explains how Stellar (“we”, “our”, “us”) collects, uses, and protects your information when using our application.

Information We Collect

When you use Stellar and connect your TikTok account, we may collect:

  • Basic TikTok profile information
  • Authorized TikTok account data provided through OAuth
  • Access token for authentication

How We Use Information

Collected data is used only to support application features, enable functionality, and improve user experience.
All data usage strictly complies with TikTok Developer Platform policies and guidelines.

OAuth / Access Token

Authentication is handled securely using TikTok OAuth. We do not collect or store user passwords. Access tokens are only used to perform actions authorized by the user.

Data Storage

User data may be stored securely to support application services and functionality.
Data is retained only for as long as necessary to provide the application functionality or as required by TikTok API policies. When data is no longer needed, it is securely deleted.

Data Sharing

We do not sell or rent user data.
User data is not shared with any third parties except TikTok services required for application functionality or when legally required.

Data Deletion & User Control

Users may revoke access or request deletion of their data by:

  • Disconnecting the application via TikTok account settings
  • Contacting us through the email below
