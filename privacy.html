Privacy Policy

Last updated: December 2025

Stellar ("we", "our", or "us") respects your privacy. This Privacy Policy explains how we collect, use, store, and protect your information when you use our application. By using Stellar, you agree to the practices described in this policy.

Information We Collect

  • TikTok account information (such as username and user ID)
  • Public profile data and authorized TikTok content
  • Basic technical information required for authentication

TikTok Data Access

Stellar only accesses TikTok data that the user explicitly authorizes through TikTok permissions. This may include:

  • Username and TikTok User ID
  • Public profile information
  • Authorized content or permitted account data

We do not access private messages, passwords, or any restricted data without user permission.

How We Use Information

  • To authenticate users via TikTok Login
  • To operate and improve application features
  • To support TikTok-related functionality
  • To comply with TikTok API policies

OAuth & Access Token Usage

Stellar uses the official TikTok OAuth authentication system to allow users to log in securely. Access tokens provided by TikTok are used only to access permitted data. We do not request or store TikTok account passwords and do not use any unofficial login methods.

Data Storage

Some data may be processed temporarily to enable the application to function properly. If any data is stored, it is protected using reasonable security measures and is not shared with unauthorized third parties. We do not sell or rent user data.

User Rights & Data Deletion

Users have full control over their data. You may:

  • Revoke Stellar access at any time through TikTok account settings
  • Request deletion of data associated with your account
  • Contact us for any privacy-related concerns

Data Sharing

We do not sell or rent user data. Data may only be shared when required to operate application functionality or comply with legal obligations.

Data Security

We implement reasonable security safeguards to protect user data from unauthorized access, misuse, or disclosure.

Policy Updates

We may update this Privacy Policy from time to time. Any changes will be posted on this page.
